This MOD divides the members profile into separate tabs, similar to the profiles on vBulletin.

Please report any bugs in the phpBB.com Support Tab or in the tracker.

Support is provided at the phpbb.com support tab and at UB's World Of Code only.

If you wish to contribute please just fork this repository, or submit a ticket and attatch a patch.
Or contribute in another way such as feature suggestions, bug reports, testing or just general feedback.
